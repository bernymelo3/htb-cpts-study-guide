## Description

## Module Objectives Checklist
| Area | Skill |
|------|-------|
| **Shell basics** | Replicate bind + reverse shell (bind on Linux, reverse on Windows). |
| **Payload basics** | Launch MSF payload, build from ExploitDB PoC, demonstrate payload-creation knowledge. |
| **Shell on Windows** | Use recon results → craft/use a payload → callback shell. |
| **Shell on Linux** | Same as above, Linux target. |
| **Web shell** | Identify common web app + language → deploy payload → browser-based shell. |
| **Spot a shell/payload** | Detect active payload/shell on a host via given info (blue-team angle). |
| **Final challenge** | Apply everything to multi-host scenario. |

## Key Takeaways
- The skills assessment requires shells on **all three** target classes: Windows, Linux, and web.
- Detection/blue-team understanding is in scope — payloads must be recognizable, not just deployable.
- Treat this section as the module's syllabus + rubric.
